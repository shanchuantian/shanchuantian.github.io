如何把有用的信息勾住

发表于 2017-06-23   |   分类于 杂谈   |  

谁都不得不向遗忘低头，没有人能记住所有的东西，而现在又是一个信息爆炸的社会。所以看的越多忘了的也越多，如果都忘了不就是把时间浪费了吗？李敖举过一个例子：

姚从吾怎么读书呢？我私下里跟我的好朋友说，姚老师看书的方法就像是一只狗熊进了玉米园，“啪”，掰下一个玉米夹在胳肢窝底下；“啪”，又掰下一个玉米夹起来带走；“啪啪啪”，忙了一晚上，最后带走的只是一个老玉米，其他努力都流失掉了。他以为夹了很多玉米，事实上只留下一个。换句话说，我的老师姚从吾教授吸收了很多东西，可是吸收以后他不能控制，不能把它们保存下来，结果再吸收新的学问时，已经吸收到的就不断流失、流失……整个循环就是一个沙漏。为什么会有这个现象？读书方法不好。读书完全凭记忆力是不行的，因为记忆会慢慢淡化。

这可以当做一个理论来解释为什么我们毕业的时候感觉学了这么多年却感觉什么也没有学到。

李敖在提到他关于读书的方法：

基本上我是一开始就买两本，看的时候按照正背面把它切下，虽然比较费钱，可是节省时间。切下来的资料怎么分类呢？我有很多夹子，在上面写上字就表示分类了。好比我写“北京大学”，夹进去的就全部是北京大学的资料。我不断用这种夹子分类，可以分出多少类呢？几千个类来，分得很细很细。一般图书馆的分类，好比哲学类、宗教类、文学类……宗教类又分佛教、道教、天主教等。我李敖分类分得比这个更细，好比“天主教类”还要细分，修女算一类，神父又算一类；神父里的同性恋算一类，还俗的又是一类。好比发生了一个跟修女同性恋有关的新闻，我要发表感想的时候，把这个夹子里的资料一打开，文章立刻写出来！一本书被我大卸八块、五马分尸完，我并不是凭记忆力去记它，而是用很细致的分类方法，很有耐心地把它钩住，放在资料夹里。这样我就把书里面的精华逮到了，这个资料就跑不掉了。

当年他身边的人一定会觉得，读个书为什么要这么麻烦？现在看来，确实有效，因为他对吸收到的信息做了索引，类似现在的打Tag（标签），我们浏览豆瓣读书的时候想看、看过一本书的时候都会对它添加标签，以后找起来就会很方便，微信也有同样的功能，我们能对朋友、收藏贴上标签，需要的时候可以很方便的找到。

所以要回答文章的标题，如何把有用的信息勾住？应该怎么做呢？我没有刻意的去搜集和打听方法，这里只把平时自己的习惯记下来，与大家交流，顺便以后改进。

刘未鹏在“学习与记忆”中也提到了李敖的读书方法，他在文章中将学习分为了两步，一是将知识转化为外显记忆，二是将外显记忆转化为内隐记忆，他也给了很多参考文献值得一读。而我这里不谈学习，只谈一些习惯。

先说说我最常用的软件：有道云笔记，坚果云，Everything，Listary。有了这些我就可以做很多事情了。有道云笔记相信大家都知道就不介绍了，坚果云是增量式的同步盘，Everything和Listary是本地文件名搜索工具。

再说说如何使用它们来帮助我提高效率。

  • 记录。用云笔记软件（同类的也有很多，像印象笔记、为知笔记，我用的是有道）记录起来非常方便，台式机、笔记本、手机，走到哪里有任何灵感都可以马上记下来，现在手机支持语音速记，走在路上不方便打字可以直接讲话，自动翻译为文字，实在方便。
  • 文件同步，随时随地学习办公。以前总是有一个问题困扰我，当我在实验室看文章时，突然要去听个报告开个会什么的，但我想在听报告的同时继续看文章怎么办？一个办法是把文章考到笔记本上，但是实在麻烦。而坚果云的出现则很好的满足了我的需求（国外有dropbox），多终端同步文件，没看完的东西、没做完的事换个地方接着做，笔记本没有电了，手机、平板看看文章也将就，这样就实现了任何时候都可以学习的诉求。
  • 收藏。我的信息获取主要是通过浏览器，另一个是微信（没有办法直接在浏览器中访问公众号，最多用电脑版的微信），有道云笔记有浏览器上的插件叫“网页剪报”功能，可以把网页收藏到云笔记中，虽然这和自己手动复制粘贴效果是一样的，但方便不少。看到有用的信息及时收藏（微信也一样）。网络传播知识的好处就是容易分享，但它的缺点也很明显，我曾经为此苦恼过好长一段时间，那就是看了不能记笔记，本地看pdf的时候我会用下划线、方框，不同颜色的文字标注、顺便再放一些与文章相差的链接、图片，这样信息就高度集中了，而浏览网页是办不到这些的。但现在至少一个缓解的方法，将网页收藏到云笔记中，在云笔记中做笔记。
  • 善用搜索。收集、记录了那么多东西记不住怎么办？解决办法是善用搜索。云笔记里面的搜索是全文搜索，也就是说搜索的时候会显示所有与关键字相关的内容，所以只有有一丁点的记忆基本上都能找到与之相关的内容。同样在本地也要善用搜索，如果不借助工具，要找某一个文件，我们只能一个盘一个盘、一个文件夹一个文件夹的去找，忘了路径就麻烦了！而windows自带的搜索实在太慢了。这时文件名搜索工具就可以登场了，Everything和Listary是都是本地文件名搜索工具，Everything的速度快得难以想象，初次使用的人都会为之惊叹，Listary也不差，它们有一些区别，大体上来讲Everything注重内容搜索，Listary（参考用法）注重对搜索到的文件的处理。有了它们，只需要零星点点的记忆，靠几个关键词就可以轻松找到想要的文件。如果想找本地文件里面的内容，Everything和Listary就没有办法了，不过也不是没有办法，用DocFetcher之类的软件就可以搜索文件里面的内容。从此，没有找不到的文件！
  • 轻整理。尽管搜索已经如此强大了，轻量的整理仍是必要的，比如给文件夹起一个实用名字，把文件放在合适的文件夹下。整理是一个再消化的过程，让知识更有体系。

最后，我时常在问自己，工具的作用到底在哪里？文豪、科学家一直都有，稍早的时代里他们连电脑都没有，但这并不影响它们成为文豪和科学家。我只能用催化剂作比方来解释，工具说到底只是工具，要做好一件事，最起码得好好去做它，工具只能让我们走得更快，却不能代替我们行走。
